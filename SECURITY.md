# Security Policy: Magnolia AI Lab

## Supported Versions
Only the latest version of the Q Protocol (v1.2+) is supported for security updates.

## Reporting a Vulnerability
The Bug Hunter agent monitors this repository for anomalies. If you discover a security vulnerability, please report it via the Identity Node at philhills.ai or by contacting the architect directly at phil@philhills.ai.

Verification of reports is handled autonomously via the Q Protocol mesh.
