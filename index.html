Common Inquiries

What is the Q Protocol?

A communication standard that replaces verbose JSON with hex coordinates. It achieves 98% token reduction and eliminates hallucinations through cryptographic receipt verification.

What is an Agent Voxel?

A compressed cognitive profile containing agent state, intent vectors, and mesh metadata. It is the atomic unit of the Q Protocol, stored as Protobuf for efficiency.
